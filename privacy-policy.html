Privacy Policy for WordPeek

Effective date: February 28, 2026

WordPeek is a Chrome extension developed by Jagroop Natt.

What WordPeek does

WordPeek shows dictionary definitions for text selected by the user on webpages.

Data Collection

WordPeek does not collect, store, or sell personal data.

How data is used

When you select a word, that word is sent to:

  • https://api.dictionaryapi.dev

This request is only used to fetch definitions.

Local Storage

WordPeek may store basic extension state (such as ON/OFF). No personal information is stored.

Third-Party Services

WordPeek uses Dictionary API:

  • https://dictionaryapi.dev
